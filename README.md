# WatchWise

An open source application that`s based on MVVM clean architecture pattern and jetpack compose that uses thetvdb.com Rest Api in order to fetch top rated and popular movies

It uses Room database to store data for watch later list.

Uses dagger hilt for dependency injection.

Uses Retrofit as a Rest client.
